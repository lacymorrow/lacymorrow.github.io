# Casper WordPress Theme

Casper is a simple, content-focused theme for writers. It was forked from the original default theme for [Ghost](https://ghost.org/),
the open source blogging platform. It's been modified to work with WordPress.

> You can find a [child theme for Casper here](https://github.com/lacymorrow/casper-child).
