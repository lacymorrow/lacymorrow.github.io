# All Seasons AC & Heating (2012)
<ThickText>[allseasonsacandhtg.com](https://allseasonsacandhtg.com/)</ThickText>

## About

> Redesigned existing website and added CMS features; Linux, Apache, MySQL, PHP

All Seasons AC & Heating is a local HVAC company in North Carolina. They had an existing website that was not mobile friendly and was not easy to update. I redesigned the website and added a CMS to make it easier for the owner to update the website.
